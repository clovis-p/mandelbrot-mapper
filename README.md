# Mandelbrot Mapper
Renders the Mandelbrot set.

## Controls
* Arrow keys to move around
* Equals key (=) : zoom in
* Minus key (-) : zoom out

Note that zooming too much breaks it and rendering is pretty slow as it is only done with the CPU. I'm getting 1 to 3 frames per second on my machine.
